Kodomi.

Project. Dec 24.

Built a versatile no-code tool empowering designers to rapidly prototype and share UI components and full websites.

Overview.

The gap between designing a component and sharing a working version of it is still measured in days. Kodomi was designed to close that gap — a tool where a designer selects a component, sets its properties, and publishes a live, interactive version in minutes without touching code.

Details.

Component Editor.
Visual

A property panel mirrors Figma's variants interface — familiar enough that any designer can use it immediately without a learning curve.

Publishing.
One tap

A shareable link generates instantly. Recipients interact with the live component in their browser — no install, no Figma account required.

Pages.
Full sites

Multiple components can be assembled into a full page or site — bridging the gap between component demo and real product preview.
