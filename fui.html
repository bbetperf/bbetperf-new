FUI.

Nugget. Dec 25.

An exploration of interface aesthetics where clarity, composition, and signal hierarchy take priority over practical constraints.

Exploration.

Fictional UI — the screens in films, games, and speculative concepts — operates under different rules. It doesn't need to be usable. It needs to communicate capability, authority, or sophistication in a single frame. This project asks: what happens when you strip that aesthetic back to its core principles and apply them rigorously?

The result is a set of interface compositions built around signal hierarchy — the idea that every element on a screen should earn its presence by communicating something specific. Noise is removed until only signal remains. What's left is interfaces that feel authoritative without being decorative.
