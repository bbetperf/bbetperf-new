Component Shortcut.

Nugget. Sep 25.

Added a custom shortcut feature for design components to reduce friction and avoid digging through the library every time.

The Problem.

Every designer who works in a large component library knows the pattern: you search, you scroll, you search again with a different term, you give up and just draw it yourself. The component is there. You use it every day. But inserting it still costs you thirty seconds and a broken flow state.

The Fix.

A keyboard shortcut mapped to a specific component. Press the shortcut, the component appears on the canvas at the cursor. No search, no scrolling, no panel interaction. For components you use more than five times a day, this saves measurable time — and more importantly, keeps you in the design rather than managing the tool.

The shortcut system supports up to twelve favorites, mapped to the function keys. It's not a new concept — code editors have done this for decades with snippets. It just hadn't been applied to design tooling in a way that felt native.
